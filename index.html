crud

Crud-functionality for mongoose and express

Install

Î» npm install crud

Usage

var express = require('express')
  , app = express.createServer()
  , models = require('./models')

require('crud')

app.crud('modelname')

REST-api

The action-mapping is inherited from express-resource and is as follows 

GET     /forums              ->  index
GET     /forums/new          ->  new
POST    /forums              ->  create
GET     /forums/:forum       ->  show
GET     /forums/:forum/edit  ->  edit
PUT     /forums/:forum       ->  update
DELETE  /forums/:forum       ->  destroy
